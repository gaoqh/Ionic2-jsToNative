# Ionic2-jsToNative
* install modules and plugins.

> $ cd Ionic2-jsToNative</br>
$ npm install</br>
$ ionic serve
</br>

* add platforms</br>

> $ ionic platform add android</br>
$ ionic platform add ios
</br>

* build</br>

> $ ionic build ios</br>
$ ionic build android
</br>

* run in ios</br>

> $ ionic run ios
</br>
